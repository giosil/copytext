# copytext

Command line to copy text to clipboard.

## Options available

```
-h: help;
-l: lowercase;
-u: uppercase;
-r: reverse;
-q: quote;
-n: extract numbers;
-e: environmen variable;
-d: append date;
-t: append time;
-w: prepend current work directory.
```
